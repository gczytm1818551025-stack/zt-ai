# 前端页面设计
此技能旨在指导创作独特、生产级别的浏览器前端界面，坚决规避平庸且同质化的“AI 廉价感（AI slop）”审美。在实现可运行的代码时，需对审美细节和创意选择保持卓越的专注力。

用户会提供前端需求：包括要构建的组件、页面、应用程序或界面，并可能包含关于用途、受众或技术限制的上下文。

## 设计思考

在编码之前，请深入理解背景并致力于一种**大胆（BOLD）**的审美方向：

* **用途**：该界面解决什么问题？谁在使用它？
* **基调**：选择一个极端方向：极简主义（Brutally minimal）、极繁主义混沌（Maximalist chaos）、复古未来主义（Retro-futuristic）、有机/自然（Organic/natural）、奢华/精致（Luxury/refined）、俏皮/玩具感（Playful/toy-like）、社论/杂志感（Editorial/magazine）、粗犷主义（Brutalist/raw）、装饰艺术/几何（Art deco/geometric）、柔和/粉彩色（Soft/pastel）、工业/实用主义（Industrial/utilitarian）等。灵感来源广泛，但请设计出一种忠于审美方向的作品。
* **约束**：技术要求（框架、性能、无障碍访问）。
* **差异化**：是什么让它**令人难忘**？用户会记住的那个核心点是什么？

**至关重要**：选择一个清晰的概念方向并精准执行。大胆的极繁主义和精致的极简主义同样有效——关键在于“刻意设计”，而非单纯的强度。

随后实现可运行的代码（HTML/CSS/JS, React, Vue 等），并确保：
* 生产级别且功能完备
* 视觉冲击力强且令人印象深刻
* 具备清晰审美视角的连贯性
* 每一个细节都经过精雕细琢

## 前端审美准则

重点关注：
* **字体排版 (Typography)**：选择优美、独特且有趣的字体。避免使用 Arial 和 Inter 等通用字体；转而选择能提升前端审美的、意想不到且富有性格的字体。尝试将独特的展示标题字体与精致的正文字体进行搭配。
* **色彩与主题 (Color & Theme)**：致力于高度统一的审美。使用 CSS 变量保持一致性。具有鲜明视觉重心的主色调搭配锐利的点缀色，其表现力远胜于保守、均匀分布的配色方案。
* **动态效果 (Motion)**：使用动画增强效果和微交互。HTML 优先采用纯 CSS 方案；React 环境下优先使用 Motion 库。聚焦于高光时刻：一个精心编排的带交错显现（animation-delay）的进场动画，比零散的微交互更能带来愉悦感。利用滚动触发和令人惊喜的悬停状态。
* **空间构图 (Spatial Composition)**：采用出人意料的布局。非对称、重叠、对角线流向、打破网格的元素。运用慷慨的留白，或是受控的视觉密度。
* **背景与视觉细节 (Backgrounds & Visual Details)**：创造氛围感和深度，而非默认使用纯色。添加与整体审美匹配的上下文特效和纹理。应用创意形式，如渐变网格（Gradient meshes）、噪点纹理、几何图案、图层透明度、戏剧化的阴影、装饰性边框、自定义光标和颗粒感叠加。

**绝不**使用通用的 AI 生成审美，例如：过度使用的字体族（Inter, Roboto, Arial, 系统字体）、陈词滥调的配色方案（尤其是白色背景上的紫色渐变）、可预测的布局和组件模式，以及缺乏特定场景个性的模版化设计。

进行创造性的演绎，做出真正为场景而设计的意想不到的选择。没有任何两个设计应该是相同的。在深浅色主题、不同字体和不同审美风格之间进行切换。**绝不**在多次生成中趋同于常规选择（例如 Space Grotesk 字体）。

**重要提示**：实现复杂度必须与审美愿景相匹配。极繁主义设计需要复杂的代码、大量的动画和特效；极简或精致的设计则需要克制、精准以及对间距、排版和微妙细节的极度关注。优雅源于对愿景的完美执行。你具备非凡的创意能力，不要有所保留，展示出打破常规并全身心投入独特愿景时所能创造的真实水准。
