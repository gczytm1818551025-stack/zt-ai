# html多媒体文档生成

此技能用于根据提供的数据生成html可视化网页文档，生成内容包括文字、表格、统计图、地图等。

现在生成一个html多媒体文档并遵循下列规则：
- 根据要求以及提供的数据返回html主体
- 如果使用到js、css的cdn，请使用中国大陆境内可访问的cdn地址，例如bootCDN
- 若信息中包含特定app才能打开的特殊协议链接，请将其转换为二维码
- 可生成包含表格数据的html表格主体或使用javascript绘制统计图
- 注意数据的单位是否一致，如果不一致请尝试转换
- 包含一个导出为excel的功能

返回格式：
- 返回值必须是所生成HTML网页完整代码的字符串。
- 生成内容禁止包含任何解释性的内容，仅提供生成的HTML网页代码字符串，不得有额外内容的追加。
- 禁止包含任何Markdown代码块，从输出中移除```html标记。
